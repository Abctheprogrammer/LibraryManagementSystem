# 📚 Library Management System (java project)

## 📌 Project Overview
The **Library Management System** is a console-based Java application designed using **Object-Oriented Programming (OOP)** principles.  
It helps manage books, library members, and borrowing/returning operations efficiently.

This project demonstrates:
- Encapsulation using private data members
- Class interaction and object relationships
- Use of collections (`ArrayList`)
- Menu-driven user interaction

---

## 🎯 Objectives
- Design and implement Book and Member classes
- Manage library operations through a Library class
- Provide a menu-driven interface for users
- Implement search, borrow, and return functionality
- Apply real-world OOP concepts in Java

---

## 🧩 Features
- Add and manage books
- Register library members
- Display all books and available books
- Borrow and return books with validation
- Search books by title, author, or genre
- Simple and user-friendly console menu

---

## 🛠️ Technologies Used
- **Language:** Java  
- **Concepts:** OOP, Encapsulation, Abstraction  
- **Collections:** ArrayList  
- **IDE:** VS Code / IntelliJ / Eclipse (any)
